# P1 - Sistema de Gestão de Biblioteca (CLI)

## Objetivo Geral
Desenvolver um **Sistema de Gestão de Biblioteca** em **Python**, via **linha de comando (CLI)**, aplicando os conceitos de **Programação Orientada a Objetos (POO)**, **persistência de dados (CRUD com SQLite)**, **boas práticas de modelagem**, **tratamento de exceções**, **testes unitários** e **versionamento com Git**
